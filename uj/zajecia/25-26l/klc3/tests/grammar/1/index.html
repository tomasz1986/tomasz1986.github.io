Korean Language Class (3) - 문법

중간고사 (2025년 12월 5일)

1. Write the following sentence in Korean using different forms.

(Please) give this letter to your father.

  • polite & formal:
  • polite & informal:
  • impolite & formal:
  • impolite & informal:

2. Correct the following sentences using proper honorific forms.

  1. 엄마가 입은 옷이 아주 예뻐.
  2. 저 사람이 우리 선생님이에요.
  3. 할머니는 다리가 많이 아프다.
  4. 김 교수가 어제 서울에 왔습니다.

3. Translate the following sentences into Korean.

  1. Mum is sleeping in the living room.
  2. I can still remember my professor's name.
  3. Cholsu's father has invited us to his house.
  4. Mr. Kim has called you to the teachers' room.

4. Which of the following sentences is not correct?

  1. 형님은 다리가 부러지셨어요.
  2. 할아버지께서는 댁이 아주 넓으세요.
  3. 선생님께서는 넥타이가 잘 어울리세요.
  4. 우리 사장님께서는 마음이 무척 착하세요.

5. Explain why the following sentence is incorrect.

할머니께서는 어머니께 김치를 드리셨어요.

6. Tick correct sentences.

  1. ☐ 문이 철칵 잠겼어요.
  2. ☐ 번개가 반짝반짝 쳐요.
  3. ☐ 시퍼런 멍이 생겼어요.
  4. ☐ 샛누런 감자가 흙에서 나왔어요.

7. Write four different adjectives for the colour “red” and put them in order from the lightest to the darkest.

Bonus. Explain the difference between 양성 모음 and 음성 모음.
