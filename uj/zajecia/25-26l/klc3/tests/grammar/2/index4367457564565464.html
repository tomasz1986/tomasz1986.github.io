Korean Language Class (3) - 문법

중간고사 (2026년 4월 23일)

1. 해가 뜨는 것을 (          ) 아침 일찍 일어났다.

  • 보아야
  • 보려고 ✓
  • 보거나
  • 보는데

2. 한국 친구 덕분에 한국 문화를 많이 (          ).

  • 알게 되었다 ✓
  • 알도록 했다
  • 알아도 된다
  • 알아야 한다

3. 동생이 점점 아버지를 (          ).

  • 닮아 간다 ✓
  • 닮기도 한다
  • 닮았나 보다
  • 닮은 적이 없다

4. 형은 차가워 (          ) 마음은 따뜻하다.

  • 보여도 ✓
  • 보여야
  • 보이다가
  • 보이든지

5. 감기약을 (          ) 열이 내렸다.

  • 먹느라고
  • 먹더라도
  • 먹을 텐데
  • 먹고 나서 ✓

6. 집에 (          ) 비가 내리기 시작했다.

  • 도착하거나
  • 도착하니까 ✓
  • 도착하거든
  • 도착하려고

7. 친구와 내가 운동장에서 축구를 (          ) 선생님이 나를 부르셨다.

  • 하거나
  • 하는데 ✓
  • 하면서
  • 하든지

8. 퇴근 후에 집에 와 보니까 동생이 여행에서 (          ).

  • 돌아와 있었다 ✓
  • 돌아오게 되었다
  • 돌아왔으면 했다
  • 돌아오도록 했다

9. 휴대 전화를 (          ) 내려야 할 역을 지나쳤다.

  • 보든지
  • 보다가 ✓
  • 보려면
  • 보고서

10. 새집으로 (          ) 가구를 새로 샀다.

  • 이사한 지
  • 이사하거든
  • 이사하려면
  • 이사하고 나서 ✓

11. 어제 본 공연은 눈물이 날 정도로 감동적이었다.

  • 날 만큼 ✓
  • 날까 봐
  • 나는 대로
  • 나는 대신에

12. 경기에서 이기고 지는 것은 연습하기에 달려 있다.

  • 연습할 따름이다
  • 연습할 모양이다
  • 연습하기 나름이다 ✓
  • 연습하기 십상이다

13. 두 사람은 오랫동안 알고 지냈지만 직장 동료 사이일 뿐이다.

  • 사이에 불과하다 ✓
  • 사이라면 좋겠다
  • 사이일지도 모른다
  • 사이라고 볼 수 없다

14. 집의 분위기가 꾸미기 나름이다.

  • 꾸밀 만하다
  • 꾸미기가 쉽다
  • 꾸밀 수도 있다
  • 꾸미기에 달려 있다 ✓

15. 동생은 차를 타기만 하면 멀미를 한다.

  • 탈 만해서
  • 타는 탓에
  • 탈 때마다 ✓
  • 타는 동안

16. 이 컴퓨터는 낡아서 수리해 봐야 오래 쓰기 어렵다.

  • 수리해 보니까
  • 수리하는 대로
  • 수리하는 바람에
  • 수리한다고 해도 ✓

17. 다음 주가 개강이니 방학도 다 끝난 거나 마찬가지이다.

  • 끝난 셈이다 ✓
  • 끝난 탓이다
  • 끝나기 마련이다
  • 끝나기 나름이다

18. 아침에 늦게 일어나는 바람에 기차를 놓쳤다.

  • 일어난 탓에 ✓
  • 일어난 김에
  • 일어나는 대신
  • 일어나는 대로

19. 정부는 일자리를 늘리고자 새로운 정책을 수립했다.

  • 늘리자마자
  • 늘리더라도
  • 늘리는 대신
  • 늘리기 위해 ✓

20. 시험이 시작되자 교실은 숨소리가 들릴 만큼 조용해졌다.

  • 들리다가
  • 들리더라도
  • 들릴 정도로 ✓
  • 들릴 때까지

2. 맞는 문법을 사용해 다음 문장들을 완성하시오.

  1. 책을 들고 다니는 걸 보니 (학생이다). (학생인 모양이네)
  2. 그는 공부는 못하는 (……) 운동은 잘한다. (반면)
  3. 기차가 부산에 (도착하다) 네 시간쯤 걸린다. (도착하려면)
  4. 철수를 (만나다) 안부를 전해 주세요. (만나거든/만나면)
  5. 일이 (끝나다) 대로 학교에 갈게요. (끝나자마자/끝나는)
  6. 그녀는 나를 못 본 (……). (척하더라/체하더라)
  7. 친구가 많이 (오다) 파티 준비를 합시다. (온다고 치고)
  8. 우리는 (울다) 서로를 바라보고 웃으면서 헤어졌다. (우는 대신에)
  9. 지은 지 20년이 넘은 집이지만 (살다). (살 만합니다)
  10. 네가 비행기를 (놓치다) 어젯밤 한숨도 못 잤다. (놓쳤을까 봐(서)/싶어(서)/해서)
  11. 거실에서 책을 (보다) 초인종이 울렸어. (보고 있는데)
  12. 수술을 하고 난 뒤 다행히 (건강하다). (건강하게 되었어요/건강해졌어요)
  13. 사람이 없으니까 집 안이 (적막하다) 합니다. (적막하기만)
  14. 날마다 (만들다) 만들었는데 오늘은 맛이 좀 이상해요. (만드는 대로)
  15. 그때가 아마 역에서 친구를 기다리던 (……). (중이었을 거야)
  16. 외국 사람이 한국어를 아무리 잘해도 (원어민이다). (원어민이라고 볼 수 없다)
  17. 칠판에 내 이름이 (쓰이다). (쓰여 있었다)
  18. 중학교 때만 (하다) 우리 집은 잘살았다. (해도/하더라도/할지라도)
  19. 그녀는 제시간에 나타나는 (……). (법이 없다)
  20. 나는 철수가 오늘 (떠나다) 몰랐어. (떠날 줄(을)/떠날지(를))
  21. 다음 주가 개강이니 방학도 다 (끝나다). (끝난 거나 마찬가지이다/끝난 셈이다/끝난 것과 같다)
  22. 눈이 많이 (오다)에 산에 갈 수 없습니다. (오는 탓에/오는 바람에)
  23. 이제는 슈퍼에서 밥(……) 팔아요. (밥까지/마저)
  24. 저 집은 음식도 (맛없다) 값이 비싸. (맛없으면서)
  25. 아이가 자는 (……) 어머니는 집안일을 끝냈어요. (자는 동안/사이)
  26. 공부가 (가다) 어려워져요. (갈수록)
  27. 나는 아침 식사를 (하다) 신문을 본다.
  28. (청소하다)에 빨래도 하자. (청소하는 김에)
  29. 그가 바보가 (아니다) 그렇게 할 수 없다. (아니고서(야/는)/아니면)
  30. 공부가 너무 많아서 머리가 터질 (……). (지경이다/터질 정도이다/터질 것 같다)
