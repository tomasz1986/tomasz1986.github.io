2. Correct the following sentences using proper honorific forms.

  1. 김 선생이 교무실로 오래.
  2. 엄마가 몹시 피곤했는지 거실에서 잡니다.
  3. 철수의 아버진 우리를 집으로 초대했어요.
  4. 나는 아직 교수의 이름을 기억하고 있어요.

3. Translate the following sentences into Korean.

  1. That is our teacher.
  2. Grandmother's legs hurt much.
  3. Prof. Kim came to Seoul yesterday.
  4. The clothes mum's wearing are very pretty.

4. Which of the following sentences is not correct?

  1. 형님은 다리가 부러지셨어요.
  2. 할아버지께서는 댁이 아주 넓어요.
  3. 선생님께서는 넥타이가 잘 어울리세요.
  4. 우리 사장님께서는 마음이 무척 착해요.

5. Explain why the following sentence is incorrect.

할아버지께서는 아버지를 모시고 공항에 가셨어요.

6. Tick correct sentences.
