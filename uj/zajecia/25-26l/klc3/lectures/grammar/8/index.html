Korean Language Class (3) - 문법

조사 (particles)

  • 일반적으로 명사, 명사 구실을 하는 말, 문장 뒤에 붙는다.
  • 격조사는 서술어의 주어와 목적어임을 나타낸다.
  • 보조사는 의미를 덧붙여 준다.
  • 접속조사는 단어와 단어를 연결해 준다.
  • 형이 집을 샀다. / Older brother bought a house.
    • 형이 - 주격 조사
    • 집을 - 목적격 조사
  • 어떻게 사느냐가 가장 중요한 문제다. / How to live is the most important question.
    • 어떻게 사느냐가 - 주격 조사
  • 중국 음식도 좋아하세요? / Do you like Chinese food as well?
    • 음식도 - 보조사
  • 너는 현교만 좋아하니? / Do you only like Hyeongyo?
    • 현교만 - 보조사
  • 저와 영미는 이번 가을에 결혼합니다.
    • 저와 - 접속조사

격조사

  • 명사/대명사/수사 - 주어와 목적어 기능
  • 동사/형용사 - 서술어 기능
  • 관형사 - 명사를 꾸미는 기능
  • 부사 - 동사와 형용사를 꾸미는 기능
  • 주격조사 - 주어 기능
  • 목적격조사 - 목적어 기능
  • 관형격조사 - 관형어 기능
  • 부사격조사 - 부사어 기능
  • 호격조사 - 독립어 기능
  • 사람들이 부산에 왔다.
  • 사람들이 부산에서 왔다.
  • 해림이(가) 노래(를) 부른다.
  • 해연이(가) 일본어(를) 공부한다.

주격조사

  • 친구들이 집으로 왔다.
  • 어떻게 사느냐가 가장 중요한 문제다.
  • 할머버지께서 시골에서 올라오셨다.
  • 학교에서 소풍을 간다.
